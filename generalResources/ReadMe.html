Welcome to Plone, a powerful and flexible open-source content management system powered by Zope and Python.

OS X version: This installer requires Mac OS X 10.5 (Leopard) or later, Intel. If you need to use a different version/processor, please install XCode and use the Unified Installer.

This Plone 4.1.5 installer creates a base installation of Python, Zope速 and Plone at /Applications/Plone. You may choose other installation destinations.

Components include:

  • Python 2.6.x, installed at InstallDestination/Python-2.6.
  • All the components of Plone 4.1.5 in a buildout cache.

Limitations

This installer contains binary components and is meant to allow a simple install of Plone without needing a binary build environment. However, it is not possible to add new binary components after the fact. If you wish to add new Plone add-ons, they must be completely Python and contain no C-language components. Fortunately, the vast majority of Plone add ons fall into this category. If you wish to be able to compile new binary components, please install XCode and use the Plone Unified Installer.

The installer creates a buildout-based installation of Plone 4.1.5 in either a stand-alone instance (default) or client-server (ZEO Cluster) configuration.

The default install will configure Zope and Plone to run in a simple, stand-alone mode. This is appropriate for testing or development. Optionally, you may choose instead to install a Zope Enterprise Objects client-server configuration, which is usually a better choice for a production server.

More information about the stand-alone versus client-server option is available at: http://plone.org/documentation/tutorial/installing-plone-3-with-the-unified-installer/to-zeo-or-not-to-zeo.

Starting Plone

After installation, instructions on starting/stopping/restarting Plone will be available at:

InstallDestination/zinstance/README.txt (for a stand-alone installation)

or

InstallDestination/zeocluster/README.txt (for a ZEO Cluster installation)

Accessing your Plone site

After starting Plone, direct your browser to http://localhost:8080/Plone to access your Plone site.

Uninstalling Plone

  • Stop all instances of Plone site
  • Delete folder InstallDestination/Plone
  • Remove startup item at /Library/StartupItems/Plone (if the startup item was selected during the install)
